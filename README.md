# FastAds

FastAds is a CLI tool to analyze competitor Meta video ads and generate structured creative insights.

## What it does

- Ingests competitor ad snapshots
- Extracts video evidence (transcript, OCR, frames)
- Breaks ads into structured components (hook, offer, angle, etc.)
- Identifies repeated patterns
- Outputs a Markdown report with insights and test ideas
